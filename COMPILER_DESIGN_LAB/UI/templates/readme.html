ðŸ“˜ Supported Functions (Extra than C++)

  • addInt(x, y): Adds two integers
  • addDouble(dx, dy): Adds two double values
  • concat(s1, s2): Concatenates two strings
  • multInt(x, y): Multiplies two integers
  • multDouble(dx, dy): Multiplies two doubles
  • maxInteg(x, y): Returns the maximum of two integers
  • maxDouble(dx, dy): Returns the maximum of two doubles
  • divInt(x, y): Divides two integers
  • divDouble(dx, dy): Divides two doubles
  • minInt(x, y): Returns the minimum of two integers
  • minDouble(dx, dy): Returns the minimum of two doubles
  • absInt(number): Returns absolute value of an integer
  • absDouble(number): Returns absolute value of a double
  • absFloat(number): Returns absolute value of a float
  • allTrue(container): Returns true if all elements in the container are true
  • toBytes(input): Converts a string to a byte vector
  • capitalize(input): Capitalizes the first letter of the string
  • title(input): Capitalizes the first letter of each word
  • swapCase(input): Swaps the case of each character
  • strCenter(input, width, fillchar): Centers the string with optional padding
  • strLjust(input, width, fillchar): Left-justifies the string with optional padding
  • strRjust(input, width, fillchar): Right-justifies the string with optional padding
  • strStartsWith(input, prefix): Checks if the string starts with the given prefix
  • strEndsWith(input, suffix): Checks if the string ends with the given suffix
  • strFind(input, substring): Finds the first index of a substring
  • strRFind(input, substring): Finds the last index of a substring
  • ord(character): Returns Unicode code point of the character
  • map(func, array): Applies a function to each element in an array
  • toBool(value): Converts an integer to boolean
  • chr(unicode): Returns the character for a Unicode value
  • complex(real, imag): Creates a complex number pair
  • divMod(dividend, divisor): Returns quotient and remainder
  • enumerate(iterable): Returns indexed pairs from an iterable
  • filter(iterable, predicate): Filters elements based on predicate function
  • toFloat(value): Converts string to float
  • isEven(number): Returns true if the number is even
  • isOdd(number): Returns true if the number is odd
  • isPrime(number): Returns true if the number is prime
  • factorial(number): Returns factorial of a number
  • gcd(a, b): Returns the greatest common divisor
  • lcm(a, b): Returns the least common multiple
  • reverseNumber(number): Returns the number with its digits reversed
  • isPalindrome(number): Checks if the number is a palindrome
  • isVowel(c): Checks if the character is a vowel
  • countDigits(number): Counts the number of digits in the number
  • sumOfDigits(number): Sums all digits in the number
  • isLeapYear(year): Checks if the year is a leap year
  • celsiusToFahrenheit(celsius): Converts Celsius to Fahrenheit
  • fahrenheitToCelsius(fahrenheit): Converts Fahrenheit to Celsius
  • strCopy(destination, source): Copies one C-string to another

int[] nums = [1, 2, 3];
bool res = allTrue(nums);               // true if all are non-zero
